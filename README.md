## Run Locally

Clone the project

```bash
  git clone https://github.com/EzeVzquez/e-commerce-react
```

Go to the project directory

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Tech Stack

**Client:** React, Firebase, NextUi, React Router
